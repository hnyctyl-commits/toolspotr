🎲 Random Number Generator

Generate random numbers within any range. Uses cryptographically secure random API.

42

True Random vs Pseudo-Random

This generator uses crypto.getRandomValues() — the same cryptographic API used for encryption keys. It's suitable for security-sensitive applications like lotteries, sampling, and password generation. For most everyday uses (dice rolls, giveaways, testing), it's more than sufficient.
