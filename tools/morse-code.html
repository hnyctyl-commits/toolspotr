📡 Morse Code Converter

Convert text to Morse code and decode Morse code back to text. Learn the classic dot-dash communication system.

Morse Code

Morse code represents letters and numbers using sequences of dots (.) and dashes (-). Invented by Samuel Morse in the 1830s for telegraph communication, it's still used today in amateur radio and aviation. SOS (... --- ...) is the universal distress signal.
