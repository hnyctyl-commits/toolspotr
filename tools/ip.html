什么是公网 IP？

公网 IP 地址是你在互联网上的唯一身份标识。通过查询 IP，你可以了解自己的网络位置信息，这对排查网络问题、检查 VPN 连接状态、确认 DNS 泄露等场景非常有用。本工具使用 WebRTC 和第三方 API 综合检测你的网络状态，数据仅在你的浏览器中展示，不会被记录或上传。
