Markdown Made Simple

Markdown is a lightweight markup language that lets you format text using plain-text syntax. Created by John Gruber in 2004, it's become the standard for documentation, README files, forum posts, and note-taking. Common syntax includes # for headings, **bold** for bold, and ``` for code blocks.
