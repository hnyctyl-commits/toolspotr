🔍 HTTP Status Code Reference

Quick reference for HTTP status codes. 1xx informational, 2xx success, 3xx redirection, 4xx client errors, 5xx server errors.

HTTP Status Code Classes

1xx — Informational: Request received, continuing process. 2xx — Success: The action was successfully received and understood. 3xx — Redirection: Further action must be taken. 4xx — Client Error: The request contains bad syntax or cannot be fulfilled. 5xx — Server Error: The server failed to fulfill an apparently valid request.
