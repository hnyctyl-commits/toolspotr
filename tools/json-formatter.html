什么是 JSON？

JSON (JavaScript Object Notation) 是一种轻量级的数据交换格式，广泛用于 API 通信、配置文件和数据存储中。格式化的 JSON 更易于人类阅读和调试，而压缩后的 JSON 更适合传输。本工具还提供实时验证功能，帮助你快速发现 JSON 语法错误。
